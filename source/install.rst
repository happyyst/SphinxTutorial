2. 安装
=====

* 安装python, 并选择按装setup tools
    + python https://www.python.org/downloads/release/python-2711/
    + setup tools http://pypi.python.org/pypi/setuptools
* 安装sphinx::

    easy_install sphinx

*  初始化sphinx项目::

    sphinx quick start

\
    根据提示一步步操作.

* 编辑index.rst

* 生成html文档::

    make html
